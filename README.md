# To-Do List Web App using ReactJS

![Todo List Web App](https://github.com/shreyamaurya029/to-do-list-reactjs-/blob/main/ss.png)

## Overview

This is a simple To-Do List Web App built using ReactJS. The app allows users to create, manage, and track their daily tasks in an organized manner. It provides a user-friendly interface to add, mark as completed, and delete tasks. The app is designed to be responsive and can be used on both desktop and mobile devices.

## Features

- Add tasks: Easily add new tasks to the list.
- Mark completed tasks: Mark tasks as completed with a single click.
- Delete tasks: Remove tasks from the list when they are no longer needed.
- Clear all completed tasks: Remove all completed tasks from the list at once for better organization.

## Installation

To run the app locally on your machine, follow these steps:

1. Make sure you have Node.js and npm (Node Package Manager) installed on your system.
2. Clone this repository to your local machine using the following command:

```bash
git clone https://github.com/your-username/todo-app.git
```

3. Navigate to the project directory:

```bash
cd todo-app
```

4. Install the required dependencies:

```bash
npm install
```

5. Start the development server:

```bash
npm start
```

6. The app will open in your default web browser at `http://localhost:3000/`.

## Usage

- **Adding a Task:** To add a new task, simply type your task in the input field at the top and press the "Enter" key or click the "Add" button.

- **Completing a Task:** When you've completed a task, click the checkbox next to the task description to mark it as completed. The task will be visually indicated as completed.

- **Deleting a Task:** To delete a task, click the "X" button on the right side of the task you want to remove. A confirmation dialog will appear before deleting the task.

- **Clearing Completed Tasks:** At the bottom of the list, there is a "Clear Completed" button. Click this button to remove all completed tasks from the list.
